Tank Game Python

To start this game you will need one Server and two Clients. To start the Server run the following command: "Python3 main.py 0.0.0.0 4444 Server"

To start the Clients run the following command in two differnt terminals: "Python3 main.py 0.0.0.0 4444 Client". As you can see the Client and Server are talking to eachother.

You can see that the Client sent the following command: {'action': 0}. This is the command to start a new game. Both Clients will need to send this command.

You can see that the Server has recieved both Clients and it has started a new game. It automatically assigned each Client a character. Client 1 is player "O" and Client 2 is player "X".

You can see that the Server recieved the following command: {'action': 1, 'row': 9, 'col': 9}. This is the command to fire.

You can see that the Server responded correctly and the * (Bullet) is in the correct position. This also shows that the other Client sent the command to fire into the box next to the first shot.
